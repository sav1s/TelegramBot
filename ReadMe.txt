Если у вас ничего не установлено выполняйте то что написанно ниже.
Если что-то есть тогда смотрите чего нету и установлюйте.

1.Вам надо скачать Python с оффициального сайта.
https://www.python.org/downloads/

2.Скачайте редактор кода, я советую Visual Code Studio
https://code.visualstudio.com/

3.Откройте cmd от имени администратора.

4.В cmd напишите "pip install pyTelegramBotAPI" 

5.Откройте файл bot.py в Visual Code Studio

6.Получите token бота в @BotFather и вставьте его в переменную token

7.В переменной id введите ваш id который вы можете получить в @getmyid_bot

8.В переменной site просто в конце ссылки мой ник QIWI замените на свой

9.В переменной channel введите ссылку на ваш канал

10.В переменной op введите @ваш_ник_телеграм

11.Теперь нажмите в Visual Code Studio, Ctrl+S, можете закрывать приложение

12.Опять открываете cmd пишите cd и копируете путь из папки в которой эта инструкция и файл bot.py и вставляете с пробелом после cd
Должно получится вот так  cd C:\TelegramBot

Ну все, если сделали все правильно бот должен работать.
